Key board

 

Key board คืออะไร

หมายถึง อุปกรณ์ป้อนข้อมูลเข้าสู่เครื่องคอมพิวเตอร์ มีลักษณะเหมือนกับแป้นของพิมพ์ดีด เพียงแต่ไม่มีก้านตัวอักษร ทำให้ไม่มีการดีดก้านพิมพ์แบบพิมพ์ดีดรุ่นโบราณ หากใช้นิ้วกดลงไปบนแป้นใดแป้นหนึ่ง อุปกรณ์นี้จะส่งสัญญาณเข้าเครื่องคอมพิวเตอร์ แล้วแสดงให้เห็นทางจอภาพ แป้นพิมพ์ของคอมพิวเตอร์จะมีแป้นต่าง ๆ เพิ่มจากแป้นพิมพ์ดีดธรรมดาหลายแป้น เช่น แป้น Esc, Return, Ctrl, Alt, Option แป้นลูกศร, แป้นกำหนดหน้าที่ ( function key ) แผงแป้นตัวเลข ฯลฯ ในปัจจุบันแป้นพิมพ์ที่ใช้เป็นมาตราฐานจะมีสูงสุดถึง 101 แป้น
